Brand signature

Morning line

A short serif epigram sits above the hero headline on rest days. It reads like a coach's margin note, not a motivational poster.

Why it works
  • Rest is explained in one sentence, so it feels intentional instead of suspiciously empty.
  • The recovery log is clearly optional and visually quieter than the hero.
  • Tomorrow is visible in one glance, which helps the user understand what today's recovery is protecting.
Guardrail

No streaks, scorecards, rings, badges, or celebratory completion language. Recovery should feel respected, not gamified.
