Brand review

Light mode should read like a private coach folio, not a flipped dashboard.

The shipped dark direction already gives FORMA a controlled athletic signature. This pass asks whether a full paper-white editorial mode can be just as premium across `Today`, `Program`, `Log`, `Intake`, and `Coach`.

Paper white canvas
Ink-black type
Accent restraint
Editorial photography
Lighter card weight
Decision

FORMA is theme-dual.

Dark should remain the launch-default signature because it carries training-time intensity and execution clarity better. But the light pass is now strong enough to be first-class, not fallback. It reads better in planning, intake, and coaching, while still keeping the same hierarchy and voice.
