Recommendation: phase ladder

The ladder is the best replacement because it gives Program a real sense of movement without duplicating the roadmap below it. It explains the current week, the reason it belongs here, and the next standard the athlete has to meet.

  • Use the ladder position to show phase, not daily turbulence.
  • Refresh the `What earned this` line from the trailing closed-week evidence window.
  • Refresh the `What earns next` line from the next visible phase gate.
  • Only move the ladder when the committed week changes or the roadmap itself regenerates.
