Today x Log hybrid-day design

Three ways to make hybrid feel coached.

Every composition treats the run and the lift as one session with one combined duration, one visible arc, and one shared rationale. Log inherits the exact same framing so the athlete feels like they are finishing the session they were promised, not switching into a second tool.

Composition 3

Weave -> Finish

Alternate run and strength inside repeating mixed blocks. This is the right shape when the session should feel continuous, dense, and athletic instead of split into two halves.

Use when the time window is tight or the coaching goal is hybrid conditioning, not max-fresh performance in either lane.
