Edge cases
  • Pause the session, and the timer pauses with it.
  • Log a set, and the current timer clears immediately.
  • Start a new rest timer, and it replaces the old one.
  • Reload or change session context, and any active timer is cleared.
Silent fallback

If alerts or speech are unavailable, the timer still runs in-app, keeps haptics when possible, and resolves to a calm `Rest finished while you were away` message on return.

System rule

This is a workout-time utility, not a reminder product. Do not bury it under weekly reminders, and do not ask for permission the first time someone taps `Rest 90s` in the gym.
