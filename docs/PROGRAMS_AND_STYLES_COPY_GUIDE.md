# Programs And Styles Copy Guide

## Purpose

Programs and Styles should sound like a premium coaching product, not a gimmick and not a legal document.

The copy should be:

- calm
- premium
- specific
- honest about source strength
- explicit about adaptation

The copy should not be:

- macho
- celebrity-obsessed
- vague startup language
- fake-certainty marketing

## Approved User-Facing Terms

- `Program`
- `Style`
- `Based on`
- `Confidence`
- `Run mostly as written`
- `Adapt to me`
- `Use as a style`

Avoid as primary UI terms:

- archetype
- planner object
- fidelity profile
- reconstruction tier
- methodology stack

## Source Labels

Approved source labels:

- `Evidence-informed default`
- `Coach-published public template`
- `Public methodology`
- `Public reconstruction`
- `Inspired by a public training archetype`

Approved confidence labels:

- `High confidence`
- `Medium confidence`
- `Low confidence`

## Naming Rules

Use the exact public name only when:

- the source is clear
- the framing is honest
- the product does not imply endorsement or official licensing

Use generalized naming when:

- sourcing is incomplete
- a celebrity identity is stronger than the evidence
- exact naming would imply precision the product cannot support

Good examples:

- `Hal Higdon-Inspired Half Marathon`
- `Golden-Era Hypertrophy`
- `Fight-Camp Lean`

Avoid:

- `Brad Pitt Fight Club Workout`
- `Arnold Exact Split`
- `Official Hal Higdon Plan` unless it is actually official

## Claim Rules

Allowed:

- `Inspired by`
- `Built around`
- `Based on public methodology`
- `Uses the structure of`
- `Borrowing the feel of`

Avoid:

- `Exactly what X did`
- `Official`
- `Verified exact plan`
- `Guaranteed transformation`

## Runtime Copy Patterns

### Program card

- `A structured multi-week plan with a clear weekly shape and progression bias.`
- `Based on: Public methodology`
- `Confidence: High confidence`

### Style card

- `A planning bias that changes the feel of the week without replacing your main goals.`
- `Based on: Inspired by a public training archetype`
- `Confidence: Medium confidence`

### Compatibility warning

- `This can work, but it needs visible adaptation.`
- `This template expects more weekly sessions than your current schedule allows.`
- `This program is built around equipment you do not have right now.`

### Activation confirmation

- `Half Marathon Base is now active in adapt to me mode.`
- `Powerbuilding Builder is selected, but the app will show visible adaptation if your current setup requires it.`

### Today explanation

- `Today's session comes from Half Marathon Base in adapted mode.`
- `Your goals still lead, but Fight-Camp Lean is shaping the feel of the week.`
- `The app is using safer fallback logic right now because the selected template is not a clean literal fit.`

### Coach explanation

- `The plan is anchored to Half Marathon Base, adapted to your current schedule.`
- `Golden-Era Hypertrophy is shaping the week, but it is not replacing your main training priorities.`
- `Recent execution has drifted enough that the app is no longer calling this strict.`

## Tone Rules For Weak Sources

When source confidence is weak:

- say `inspired by`
- say `style direction`
- say `not an exact public routine`

Do not:

- imply endorsement
- imply exact reconstruction
- hide low confidence behind slick branding
