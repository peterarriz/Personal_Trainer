# Theme Distinctness And Discoverability Spec

## Theme System
- FORMA ships 10 curated identities.
- Distinctness must be perceptual, not only token-level.
- Theme cards should preview real surface, accent, and typography differences.

## Mode Rules
- `System` follows the OS preference.
- `Dark` and `Light` remain intentionally different presentations, not minor tint flips.
- Light mode must stay premium and readable.

## Discoverability
- Appearance lives in Settings.
- Theme previews should help users understand the difference before selection.
- Account, plan management, and advanced settings should remain separate from theme selection so the page does not collapse into one giant mixed admin panel.
