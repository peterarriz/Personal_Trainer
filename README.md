# Personal Trainer

AI-powered half marathon training dashboard with strength, nutrition, and Achilles prehab tracking.

## Stack
- React 18 (inlined, no CDN)
- Supabase (persistent storage)
- Anthropic API (AI coaching + dynamic plan adjustments)

## Workflow

### Making changes
1. Edit `src/trainer-dashboard.jsx` (or ask Claude to update it)
2. Run `npm run build` — compiles JSX → `index.html`
3. `git add . && git commit -m "your message" && git push`
4. Vercel auto-deploys in ~30 seconds

### First time setup
```bash
npm install
npm run build
```

### Local preview
```bash
npm run dev
# Open http://localhost:3000
```

## Files
- `src/trainer-dashboard.jsx` — source of truth, edit this
- `index.html` — compiled output, auto-generated by build script
- `scripts/build.js` — build script (JSX → HTML)
- `scripts/react.min.js` — React 18 (inlined at build time)
- `scripts/react-dom.min.js` — ReactDOM 18 (inlined at build time)
- `vercel.json` — Vercel build config

## Adding to iPhone home screen
1. Open your Vercel URL in Safari
2. Tap Share → Add to Home Screen
3. Done — works like a native app
